list
===

### 说明
该参数指定kplayer需要推流的目标媒体资源文件。由`数组`类型构成。
参数均支持相对路径与绝对路径。
::: tip 提醒
请填写完整可访问的文件路径，不支持目录、通配符、缺少后缀名等路径
:::

### 类型
| 数据类型 | 是否必填 | 默认值 |
|---|---|---|
| 数组 | 是 | - |

### 示例
```yaml {5,6}
version: "1.0"
player:
  media:
    list:
      - "/path/example1.mp4"
      - "/path/example2.mp4"
```
