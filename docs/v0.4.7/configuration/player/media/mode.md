mode
===

### 说明
设置播放列表播放模式

### 类型
| 数据类型 | 是否必填 | 默认值 | 可选值 | 描述 |
|---|---|---|---|---|
| 枚举 | 否 | list | list| 顺序播放 |
|  |  |  | random | 随机播放，不保证随机完整性 |
|  |  |  | queue | 队列模式，资源播放完成做在列表中删除处理，直至保留最后一个资源文件 |

### 示例
```yaml {4}
version: "1.1.0"
player:
  media:
    mode: "list"
```
